
# 타입스크립트로 todolist 만들기
`TypeScript`와 `Redux`를 연습하기 위한 작업물입니다.<br>

`Storybook`을 이용해 컴포넌트를 테스팅 및 관리하였고, `Redux`로 전역적으로 상태를 관리하였습니다.
